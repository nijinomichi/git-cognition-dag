# Discussion (draft)

The empirical falsification conditions stated earlier ask whether the model is
supported by data. This section asks a prior and different question: under what
conditions does the mapping itself cease to hold, so that G = (V, E, P, H)
stops standing for cognition and reverts to being a bare data structure. These
are theoretical breakdown modes, not experimental outcomes. Stating them is
what closes the model: a scientific account is complete only when its
conditions of validity and its conditions of collapse are both specified.

Four breakdown modes recur for any Git-as-cognition account.

## 1. Representational failure

The mapping collapses when Git structure no longer corresponds to cognition.
This appears when history stops carrying meaning: bulk auto-generated or noise
commits, changes that are log compression rather than state transitions, or
pointers (P) that lose semantic stability so that HEAD no longer denotes a
cognitive position. At that point the equation V, E, P, H approx cognition
simply fails, because the graph records activity without recording
understanding.

## 2. Coordination collapse

The mapping collapses when the same G is read as different Gs by different
agents. This appears when branch and merge semantics are interpreted
differently across a team, when AI-generated history structures diverge from
human interpretive models, or when conflict resolution is handled
probabilistically rather than by meaning. Git then stops being shared memory
and becomes a mere communication log: structurally intact, but no longer
cognition-bearing infrastructure because there is no shared object to reason
over.

## 3. Temporal degeneration

The mapping collapses when history operations (H) lose their proper
irreversibility, in either of two directions. Under excess irreversibility,
reset and rewrite are so frequent that nothing durable remains. Under excess
reversibility, everything is rebaseable and no past is ever fixed. Both remove
the thickness of time, and without a stable temporal layer the experiential
structure that the cognition claim depends on cannot form.

## 4. External substitution failure

The mapping collapses when Git ceases to augment cognition and instead fully
replaces it. This appears when humans no longer read history and only an AI
interprets meaning, when sense-making migrates from V, E, P, H into the
internal state of a language model, or when the repository becomes
unread infrastructure. At that moment the model can no longer refer to itself:
Git is no longer cognition made external and inspectable, but an opaque data
structure whose meaning lives elsewhere.

## The one-line failure condition

Across all four modes the theory breaks the same way: the moment Git stops
being a history that carries meaning, V, E, P, H revert to being mere
structural symbols.

## Why this strengthens rather than weakens the account

These conditions are not concessions. A model that can specify exactly how and
where it breaks is a theory; a model that cannot is a metaphor. By pairing the
validity conditions of the formal model with these four collapse modes and the
empirical falsification conditions, the account is closed on both sides. This
is also the boundary at which the framing earns the label of theory rather
than philosophy: it holds only where Git remains meaningful history, and it
says plainly where that ceases to be true.
